# 🛒 E-commerce Sales & Customer Analytics (SQL Project)

## 📌 Project Overview

This project focuses on analyzing an e-commerce dataset using SQL to extract meaningful business insights related to customer behavior, sales performance, and product trends.

The goal of this project is to demonstrate strong SQL skills including database design, data querying, and analytical thinking for real-world business scenarios.

---

## 🗂️ Dataset

The dataset used in this project is from Kaggle:

🔗 https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

It contains information about:

* Customers
* Orders
* Order Items
* Products

---

## 🧱 Database Schema

The project is built using a relational database with the following tables:

* **customers** → Customer details
* **orders** → Order information
* **order_items** → Products included in each order
* **products** → Product details

These tables are connected using primary and foreign keys to simulate a real-world e-commerce database.

## 📊 Key Business Insights

### 1️⃣ Total Orders per Customer

Identified how many orders each customer placed to understand repeat purchasing behavior.

### 2️⃣ Top Customers

Analyzed customers with the highest number of orders.

### 3️⃣ Revenue per Customer

Calculated total spending by each customer using transaction data.

### 4️⃣ Most Sold Products

Identified products with the highest sales volume.

### 5️⃣ Top Revenue-Generating Categories

Analyzed which product categories contribute the most to overall revenue.
